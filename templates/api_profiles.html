How API Connections Work

1

Find Your API Credentials

Log in to your SMM panel account. Look for an API section in your account settings to get your API URL and key.

2

Add a New API Profile

Click the "Add New API" button and enter your API URL, API key, and a memorable name for this connection.

3

Test Your Connection

After adding, test your connection to make sure it works properly with your SMM panel.

4

Start Automating

Once connected, you can use this API profile to set up automated orders and jobs.
